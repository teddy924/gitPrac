Vim은 강력한 텍스트 편집기입니다.
다양한 운영 체제에서 사용할 수 있으며, 프로그래머와 시스템 관리자가 선호하는 도구 중 하나입니다.

1. Vim의 주요 특징:
   - 경량 텍스트 편집기.
   - 대부분의 Linux/Unix 시스템에 기본 설치.
   - 빠르고 강력한 단축키 제공.

2. Vim 모드:
   - Normal 모드: 명령 입력에 사용.
   - Insert 모드: 텍스트 입력에 사용.
   - Visual 모드: 텍스트 선택에 사용.
   - Command-Line 모드: 저장, 종료, 검색 명령에 사용.

3. 자주 사용하는 명령어:
   - 저장하기: :w
   - 종료하기: :q
   - 저장 후 종료하기: :wq
   - 저장하지 않고 종료하기: :q!

4. 이동 단축키:
   - h: 왼쪽으로 한 칸 이동
   - l: 오른쪽으로 한 칸 이동
   - j: 아래로 한 줄 이동
   - k: 위로 한 줄 이동

5. 텍스트 삭제:
   - x: 커서 위치의 문자 삭제
   - dd: 현재 줄 삭제
   - d$: 커서부터 줄 끝까지 삭제

6. 텍스트 복사와 붙여넣기:
   - yy: 현재 줄 복사
   - p: 복사한 내용 붙여넣기

7. 검색:
   - /단어: 단어 검색
   - n: 다음 검색 결과로 이동
   - N: 이전 검색 결과로 이동

8. 기타 유용한 팁:
   - u: 작업 취소
   - Ctrl + r: 작업 다시 실행
   - gg: 파일의 처음으로 이동
   - G: 파일의 끝으로 이동
   - :set number: 줄 번호 표시
